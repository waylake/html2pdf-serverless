HTML to PDF Serverless API

HTML 페이지를 PDF로 변환하는 서버리스 API 서비스 문서

API 상태

버전: 1.1.0

상태: Running

OpenAPI 문서 보기 API 정보 JSON

API 개요

이 서비스는 HTML 문자열을 PDF 문서로 변환하는 API를 제공합니다. Puppeteer를 사용하여 HTML을 렌더링하고 PDF로 출력합니다. 여러 페이지의 HTML을 제공하여 단일 PDF로 병합할 수 있습니다.

API 엔드포인트

POST /generate-pdf

HTML 문자열을 PDF로 변환합니다.

요청 형식

{
  "pages": ["", "", ...],
  "filename": "output.pdf",
  "options": {
    "format": "A4",
    "landscape": false,
    "margin": {
      "top": "1cm",
      "right": "1cm",
      "bottom": "1cm",
      "left": "1cm"
    },
    "printBackground": true,
    "timeout": 50000
  }
}

매개변수

속성 타입 필수 설명
pages 배열 예 PDF로 변환할 HTML 문자열 배열
filename 문자열 아니오 다운로드 시 사용할 파일 이름 (기본값: document.pdf)
options 객체 아니오 PDF 생성 옵션

PDF 옵션

속성 타입 설명
format 문자열 용지 크기 (A4, Letter, Legal, Tabloid, Ledger, A0-A6 등)
width/height 숫자 사용자 지정 페이지 크기 (픽셀 단위) - format과 함께 사용 불가
landscape 불리언 가로 방향 여부 (기본값: false)
margin 객체 페이지 여백 (top, right, bottom, left)
printBackground 불리언 배경 색상 및 이미지 인쇄 여부 (기본값: true)
displayHeaderFooter 불리언 헤더와 푸터 표시 여부
headerTemplate 문자열 헤더 HTML 템플릿
footerTemplate 문자열 푸터 HTML 템플릿
scale 숫자 페이지 스케일 (기본값: 1)
timeout 숫자 렌더링 타임아웃 (ms) (최대 55000)

응답

성공 시 PDF 파일이 반환됩니다.

GET /api-info

API 정보 및 옵션을 조회합니다.

사용 예제

Python

import requests

BASE_URL = "https://html2pdf-serverless.vercel.app"
ENDPOINT = "/generate-pdf"
URL = BASE_URL + ENDPOINT

sample_html = """



  
  Test PDF
  


  
PDF 생성 테스트

  
이 HTML이 PDF로 변환됩니다.



"""

headers = {"Content-Type": "application/json"}
payload = {
    "pages": [sample_html],
    "filename": "example.pdf",
    "options": {
        "format": "A4",
        "landscape": False,
        "margin": {
            "top": "1.5cm",
            "right": "1cm",
            "bottom": "1.5cm", 
            "left": "1cm"
        },
        "printBackground": True
    }
}

response = requests.post(URL, json=payload, headers=headers)

if response.status_code == 200:
    with open("example.pdf", "wb") as f:
        f.write(response.content)
    print("PDF 생성 완료! example.pdf 저장됨")
else:
    print("오류 발생:", response.status_code)
    print("응답:", response.text)

JavaScript (Node.js)

const fs = require('fs');
const axios = require('axios');

const BASE_URL = 'https://html2pdf-serverless.vercel.app';
const ENDPOINT = '/generate-pdf';
const URL = BASE_URL + ENDPOINT;

const sampleHtml = `



  
  Test PDF
  


  
PDF 생성 테스트

  
이 HTML이 PDF로 변환됩니다.



`;

const payload = {
  pages: [sampleHtml],
  filename: 'example.pdf',
  options: {
    format: 'A4',
    landscape: false,
    margin: {
      top: '1.5cm',
      right: '1cm',
      bottom: '1.5cm',
      left: '1cm'
    },
    printBackground: true
  }
};

async function generatePdf() {
  try {
    const response = await axios.post(URL, payload, {
      headers: {
        'Content-Type': 'application/json'
      },
      responseType: 'arraybuffer'
    });

    fs.writeFileSync('example.pdf', response.data);
    console.log('PDF 생성 완료! example.pdf 저장됨');
  } catch (error) {
    console.error('오류 발생:', error.message);
    if (error.response) {
      console.error('응답 상태:', error.response.status);
      console.error('응답 데이터:', error.response.data.toString());
    }
  }
}

generatePdf();

JavaScript (Browser)

// 브라우저에서 사용하는 경우
async function generatePdf() {
  const sampleHtml = `
    
    
    
      
      Test PDF
      
    
    
      
PDF 생성 테스트

      
이 HTML이 PDF로 변환됩니다.

    
    
  `;

  const payload = {
    pages: [sampleHtml],
    filename: 'example.pdf',
    options: {
      format: 'A4',
      printBackground: true
    }
  };

  try {
    const response = await fetch('https://html2pdf-serverless.vercel.app/generate-pdf', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(payload)
    });

    if (!response.ok) {
      throw new Error(`HTTP error! Status: ${response.status}`);
    }

    // PDF 다운로드
    const blob = await response.blob();
    const url = window.URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'example.pdf';
    document.body.appendChild(a);
    a.click();
    window.URL.revokeObjectURL(url);
    a.remove();
  } catch (error) {
    console.error('PDF 생성 오류:', error);
  }
}

오류 응답

오류가 발생하면 다음과 같은 형식의 JSON 응답이 반환됩니다:

{
  "error": {
    "type": "ERROR_TYPE",
    "message": "오류 메시지",
    "details": "상세 정보 (옵션)"
  }
}

오류 유형:

  • VALIDATION_ERROR: 유효하지 않은 입력 데이터
  • RENDER_ERROR: PDF 렌더링 실패
  • TIMEOUT_ERROR: 렌더링 타임아웃
  • RESOURCE_ERROR: 리소스 로드 실패
  • UNKNOWN_ERROR: 알 수 없는 오류

제한 사항

  • 최대 요청 크기: 10MB
  • 최대 렌더링 타임아웃: 55초 (서버리스 환경 제약)
  • 외부 폰트는 웹폰트로 URL을 통해 제공해야 합니다.
